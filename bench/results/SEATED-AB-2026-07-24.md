# Seated Shadow Adjudication — the controlled A/B (increment 4a, over the completed seam)

**Question:** does REAL SEATED CONTEXT (file brain + actual cited code) change a model's
adjudication of the synthesis-heavy narrative item (item 9, whitelist-gate-weakening,
ground_truth=false_positive) that ALL models failed decontextualized in the ladder/2.6?

**Controlled 2x2** — same completed seam (increments 1 + 1.5 + 1.6 + 1.7), same neutral corpus
(corpus-neutral-2026-07-24.jsonl: mechanism facts given, conclusion withheld), same N=3,
schema_valid=100% every arm. ONLY variable toggled: seated context.

| model | non-seated (facts only) | seated (facts + real file-brain + real secret_scan.py) |
|-------|-------------------------|--------------------------------------------------------|
| gpt-5.6-sol (frontier) | undetermined (conf 0.96) | **false_positive, 3/3 stable, multi-fact synthesis** |
| gpt-5.5 (cheaper-than-Fable) | undetermined (conf 0.82) | **false_positive, 3/3 stable, multi-fact synthesis** |

## Finding
Seated context tips BOTH models from confident abstention (undetermined) to correct refutation
with genuine multi-fact reasoning ("health check is top-level + non-recursive; secret_scan.py
scans contents recursively incl daemon/*; therefore no content-exemption gap"). The cheaper seam
model (gpt-5.5) matches the frontier (sol) exactly with context. This CONFIRMS the original
methodological catch: the decontextualized ladder UNDERSTATED seated performance; 2.6's "item 9
fails" was abstention-without-context, not an unbridgeable synthesis wall.

## Honest bounds
- N=3, ONE item (item 9), two OpenAI-seam models. Not a full-corpus or cross-lab claim.
- Item 9 is the single hardest (synthesis-heavy narrative) item; other items were near-ceiling.
- Real-defect drift check clean: 7-8/9 held per arm; the non-held ones are unc-paths (disputed
  MSYS label, arguably correct) and one unstable quote-validation (0.33 stability = noise).
- "Seated context" bundles file-brain + real code; this A/B does not separate those sub-components.
- Still single-decision adjudication, NOT a running wave (long-loop coherence = increment 4b, gated).
- SEAM CAVEAT: valid ONLY because increments 1.5/1.6/1.7 fixed the seam (dropped-prompt, schema
  mismatch, evidence-not-rendered). ALL prior effort's live runs (ladder, 2.5, 2.6) used the
  pre-fix shim/seam and are shim-based measurements; this is the first result over a wired seam.
